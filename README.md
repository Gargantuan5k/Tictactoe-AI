## Usage

1. Clone the Git repository:

  ```sh
  git clone https://github.com/Gargantuan5k/Tictactoe-AI
  ```
2. CD into the src directory

```sh
cd Tictactoe-AI/src
```

3. Run the main.py file

```sh
python3 main.py
```

OR (Windows only)
1. Download the main.exe file from assets
2. Open the main.exe file
